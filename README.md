**XEditor** (formerly XNote) is a lightweight, privacy-focused desktop note editor built with Tauri (Angular + Rust).

## Core characteristics
- Local-first note-taking (no cloud dependency)
- Markdown editor with live preview (split view)
- Plain text + formatted editing
- Customizable UI
    - Font size
    - Font color
    - Autosave behavior
- Fast, native performance (Rust backend)
- Minimal, distraction-free interface
- Positioning (short description)

XEditor is a fast, local-first Text/Markdown note editor with live preview and customizable writing experience.
